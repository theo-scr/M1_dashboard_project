Développement d'un système d’aide à la décision sur le thème des véhicules à hydrogène. 

Utilisation : 
	1) Ouvrir l'application Jupyter
	Attention : il faut exécuter la commande suivante dans la console qui démarre Jupyter:
		export LD_LIBRARY_PATH=/opt/oracle/instantclient_19_5:$LD_LIBRARY_PATH
	
	2) Ouvrir l'un des trois fichiers : "scrapingV1.4", "insert_database_V0.4" ou "data_visualization_V0.3"
	
	3) Exécuter toutes les cellules. 


Contenu:
	- Le dossier 'scraping' contient tous les fichiers nécessaires à l'extraction d'informations : un notebook et trois DataFrames. 
	- Le dossier 'database' contient tous les fichiers nécessaires à l'administration de la base de données : un script SQL de création, un notebook d'insertion et un script d'interrogation des données.
	- Le dossier 'visualisation' contient tous les fichiers nécessaires à la valorisation des données : un notebook et l'ensemble des graphiques qu'il produit. 
	- Le fichier 'groupe2_rapport_vehicules_hydrogenes_V8.pdf' contenant le rapport détaillé du projet.
	- Le fichier 'README.txt' permettant de présenter les différents dossiers et fichiers du projet.
